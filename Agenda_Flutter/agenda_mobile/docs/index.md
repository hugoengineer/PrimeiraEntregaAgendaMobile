# 📅 Agenda Inteligente

Bem-vindo à documentação oficial do projeto **Agenda Inteligente**. Este aplicativo foi desenvolvido para ajudar usuários a organizar tarefas e compromissos de forma simples e intuitiva.

## 🚀 Sobre o Projeto
O objetivo principal é oferecer uma experiência fluida de agendamento, com notificações em tempo real e sincronização local/nuvem.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Dart/JavaScript
- **Framework:** Flutter/React Native
- **Banco de Dados:** SQLite / Firebase
- **Containerização:** Docker

## 📱 Funcionalidades Principais
- [ ] Sistema de login.

- [ ] Recuperarção de senha
- [ ] Visualização dos eventos diários.

## 📦 Como rodar o projeto
Para rodar este app localmente, siga os passos abaixo:
1. Clone o repositório.
2. Configure as chaves de API no arquivo `.env`.
3. Execute `docker-compose up` para ver esta documentação!
